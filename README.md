# Adesso-Video-Capture-Express
Windows 10 Drivers for Adesso Video Capture Express USB 2.0 Device aka `USB 28286 Device`

Driver version 5.13.1218.1924 dated 7/24/2014

Recovered from a Windows 10 64-bit IoT LTSC install on 1 Feb 2026 using the command line `dism /online /export-driver /destination:L:\Drivers`

Hardware IDs
```
USB\VID_1B80&PID_E607&REV_0100&MI_00
USB\VID_1B80&PID_E607&MI_00
```

Posted here as drivers are unavailable from the manufacturer, drivers remain (c) their respective owners.
